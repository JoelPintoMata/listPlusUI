# What is

## Run
```
$ npm start
```

## Endpoints

Endpoint | Example | Description
-------- | ------- | -----------
http://<server_name>:<port>/myLists | http://localhost:4200/myLists | My lists available
http://<server_name>:<port>/myList/<my_list_id> | http://localhost:4200/myList/1 | My list details
